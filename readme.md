# Currency Converter React App

A simple **Currency Converter App** built with **React.js** that lets you convert values between different currencies using real-time exchange rates.  
This project helped me learn how to fetch data from api

---

##  What I Learned

- How to use **React Hooks** (`useState`, `useEffect`)
- How to build and use a **custom hook** (`useCurrencyInfo`)
- Fetching and handling **API data** in React
- Working with **props** and **component reusability**
- Styling with **Tailwind CSS**
